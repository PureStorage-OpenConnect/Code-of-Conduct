# Everpure Open Connect Contributor Code-of-Conduct
Code of Conduct for all Everpure Open Connect projects

Refer to [Open Source Contribution Code of Conduct.md](https://github.com/PureStorage-OpenConnect/Code-of-Conduct/blob/main/Open-Source-Contribution-Code-of-Conduct.md)

This code of conduct applies to all of the projects under the Everpure Open Connect organization on GitHub and the Everpure Open Source community at large.

If you have suggestions to improve this code of conduct, please submit an issue or PR.
